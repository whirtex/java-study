# Sistema de Controle de Medicamentos

Projeto desenvolvido como atividade prática da disciplina de Orientação a Objetos em Java.

## Conceitos aplicados
- Abstração
- Encapsulamento

## Funcionalidades
- Cadastro de medicamentos com validação de dados
- Entrada de estoque
- Venda de unidades com verificação de estoque
- Reajuste de preço por percentual

## Como executar
1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`

## Tecnologias
- Java
