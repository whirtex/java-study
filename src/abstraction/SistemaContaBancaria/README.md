# Sistema de Conta Bancária

Projeto desenvolvido como atividade prática da disciplina de Orientação a Objetos em Java.

## Conceitos aplicados
- Abstração
- Encapsulamento

## Funcionalidades
- Cadastro de conta com validação de dados
- Depósito de valores
- Saque com verificação de saldo
- Consulta de saldo

## Como executar
1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`

## Tecnologias
- Java
