# Sistema de Vaga de Estacionamento

Projeto desenvolvido como atividade prática da disciplina de Orientação a Objetos em Java.

## Conceitos aplicados
- Abstração
- Encapsulamento

## Funcionalidades
- Cadastro de vagas com validação de dados
- Ocupação de vaga com verificação de tipo de veículo
- Liberação de vaga com cálculo do valor a pagar
- Controle de status: Livre e Ocupada
- Validação de tipos: MOTO, CARRO e CAMINHAO

## Como executar
1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`

## Tecnologias
- Java
