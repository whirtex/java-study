# Sistema de Reserva de Passagens Aéreas

Projeto desenvolvido como atividade prática da disciplina de Orientação a Objetos em Java.

## Conceitos aplicados
- Abstração
- Encapsulamento

## Funcionalidades
- Cadastro de passagens com validação de dados
- Confirmação de reserva
- Cancelamento de reserva com aplicação de multa de 15%
- Upgrade de classe com valor adicional
- Controle de status: CONFIRMADA, CANCELADA e AGUARDANDO

## Como executar
1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`

## Tecnologias
- Java
