# Sistema de Gerenciamento de Alunos de Academia

Projeto desenvolvido como atividade prática da disciplina de Orientação a Objetos em Java.

## Conceitos aplicados
- Abstração
- Encapsulamento

## Funcionalidades
- Cadastro de alunos com validação de dados
- Registro de aulas com verificação de mensalidade
- Renovação e suspensão de mensalidade
- Validação de planos: BASICO, INTERMEDIARIO e PREMIUM

## Como executar
1. Clone o repositório
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe `Main.java`

## Tecnologias
- Java
